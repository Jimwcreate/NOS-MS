## How to Use

### 1. Access User Management
```
Admin Dashboard → User Management (sidebar)
```

### 2. Manage Students
- Select "Students" from dropdown
- Search by name, email, or roll number
- Click "Reset" to reset password to 12345678
- Click "Delete" to remove student account

### 3. Manage Teachers
- Select "Teachers" from dropdown
- Search by name or email
- Click "Reset" to reset password
- Click "Delete" to remove teacher account

### 4. Manage Admins
- Select "Admins" from dropdown
- View all admin accounts
- Reset passwords as needed
- Delete other admins if necessary

## API Endpoints Reference

### Delete User
```javascript
POST /api/DeleteUser
Header: Authorization: Bearer <admin_token>
Body: {
  "userId": "user_id",
  "userType": "student|teacher|admin"
}
```

### Reset Password
```javascript
POST /api/ResetUserPassword
Header: Authorization: Bearer <admin_token>
Body: {
  "userId": "user_id",
  "userType": "student|teacher|admin"
}
```

### Get User Summary
```javascript
GET /api/UsersSummary/:adminId
Header: Authorization: Bearer <admin_token>
```

## Example Workflows

### Reset Forgotten Password
```
1. Admin: User Management
2. Admin: Selects "Students"
3. Admin: Searches for student name
4. Admin: Clicks "Reset" button
5. Admin: Confirms in dialog
6. Student: Receives notification
7. Student: Logs in with 12345678
8. Student: Changes password
```

### Remove Inactive User
```
1. Admin: User Management
2. Admin: Selects user type
3. Admin: Finds user in list
4. Admin: Clicks "Delete" button
5. Admin: Reviews details
6. Admin: Confirms deletion
7. System: Removes user from database
8. User: Cannot login anymore
```
